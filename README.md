# 🎮 JURIS QUEST

> ⚖️ A Jornada da Aprovação — Concurso Soldado PM

Um jogo gamificado de estudos para concursos policiais militares, cobrindo **Direito Constitucional, Administrativo e Penal** com mais de 500 questões reais das principais bancas examinadoras do Brasil.

🌐 **Jogue agora:** _coloque aqui o link do GitHub Pages depois de publicar_

---

## ✨ Features

- 🗺️ **30 dias de batalha** intercalados (Const → Adm → Penal), cada um um boss de 30 questões
- 🏆 **7 bancas integradas:** CESPE/CEBRASPE, FGV, FCC, VUNESP, IBFC, AOCP e Quadrix
- ⚠️ **Emboscadas surpresa** entre os dias (30% de chance) com 10 questões aleatórias
- 🧠 **Sistema de Revisão Espaçada (SRS)** — questões erradas voltam em 1d → 3d → 7d → 21d → 60d
- 🎯 **Caçada à Pegadinha** — 10 armadilhas clássicas CESPE
- ⏱️ **Simulado cronometrado** — 20 questões, 50 minutos
- 📊 **Heatmap de fraquezas por matéria** com recomendação personalizada
- 🃏 **20 cartas colecionáveis** (súmulas, leis, jurisprudências)
- 🏆 **Ranking público opcional** — compita com outras concurseiras
- ⏳ **Cronograma adaptativo** baseado na data da prova
- 👮‍♀️ **12 patentes** de Recruta a Capitão
- 🔊 **Som retrô 8-bit** sintetizado
- 👤 **Avatar customizável** (12 ícones × 6 cores)
- ☁️ **Salvamento na nuvem** com Supabase — acesse de qualquer dispositivo

## 🎨 Design

Visual arcade neon (CRT, scanlines, pixel corners, glow). Paleta Fluently em mauve/burgundy/rosé. Tipografia Press Start 2P + Orbitron + Rajdhani.

## 🛠️ Tecnologia

100% client-side, sem build system:
- HTML + CSS + JavaScript puro
- [Supabase](https://supabase.com) para autenticação e banco PostgreSQL na nuvem
- Web Audio API para sons sintetizados
- Google Fonts para tipografia

## 🚀 Como usar

1. Acesse o link do site
2. Crie sua conta com email e senha
3. Complete o onboarding (cargo, banca, data da prova)
4. Comece pelo Dia 1 e siga sua jornada
5. Marque "Aparecer no ranking" se quiser competir

## 📚 Conteúdo coberto

**Constitucional:** Princípios fundamentais, Art. 5º (direitos pessoais e remédios), Segurança Pública (art. 144)

**Administrativo:** LIMPE, Poderes administrativos, Atos administrativos, Servidores e Improbidade (Lei 8.429/92 + Lei 14.230/21)

**Penal:** Princípios, Aplicação da lei (tempo/lugar), Crimes contra pessoa/patrimônio/Administração, Lei de Drogas (11.343/06), Lei Maria da Penha (11.340/06), Estatuto do Desarmamento (10.826/03), Tortura (9.455/97), Hediondos (8.072/90), ECA (8.069/90) e Direitos Humanos
